### python version
    - 3.12.3
### 가상환경 설정
    - python -m venv .venv
### 가상환경 on
    - source .venv/bin/activate
### 패키치 설치
    - pip install -r requirements.txt
### streamlit(frontend) 실행
    - cd frontend
    - streamlit run frontend.py
### fastapi(backend) 실행
    - cd backend
    - python backend.py
    
- backend 폴더에 .env 파일 생성하셔야 합니다.
